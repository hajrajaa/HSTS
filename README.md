# HSTS - High School Test System 


HSTS (High School Test System) is a Software soultion designed to makes managing exams in high schools easier.
It uses JavaFX for a dynamic user interface, CSS for aesthetics, and MySQL for database storing. The system has different features for students, teachers, and principals to help them with exams system.
The project was developed as part of the Software Engineer course at University of Haifa .  


## Running
1. Clone the repostory on your *local Machine*
2. Run Maven install **in the parent project**.
3. Run the server using the exec:java goal in the server module.
4. Run the client using the javafx:run goal in the client module.
